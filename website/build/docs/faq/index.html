FAQ (Frequently Asked Questions)

Is PixiEditor free?#

Yes! The project is entirely open-source under MIT License.

Can I use art made in PixiEditor in commercial projects?#

Yes! Everything you make in PixiEditor is yours!

What platforms does PixiEditor support?#

Currently, you can only run the editor on Windows, however, we have plans for Linux and macOS versions.

Is PixiEditor related to Pixi.js?#

Not at all! Pixi.js is a TypeScript/JavaScript rendering library, while PixiEditor is a pixel-art editor built entirely on .NET and C#, it's just a coincidence of names.
